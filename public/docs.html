REST API v1.3

Panduan Integrasi API FazzPay

Gunakan API resmi FazzPay untuk mengotomatiskan pembuatan dokumen kwitansi transfer beresolusi ultra-tinggi dalam format PNG biner asli yang digambar langsung di backend Rust-powered Node Canvas.

Penggunaan API Key Secara Rahasia

Gunakan Kunci API yang sah di lingkungan back-end atau server terenkripsi Anda. Jangan menyebarkan Kunci API secara publik di dalam script HTML klien untuk mencegah penyalahgunaan.

Definisi Endpoint

POST /api/generate

Menerima payload bertipe JSON dan secara langsung merespons dengan konten dokumen gambar biner PNG berkualitas tinggi.

Parameter Payload (JSON)

Kolom Tipe Wajib Deskripsi
apiKey string Wajib Token otorisasi keamanan Anda (Kunci standar: "fazzganzz").
amount string Wajib Nilai nominal transaksi (Format string, Contoh: "500.000").
productName string Wajib Nama barang, langganan server, atau deskripsi jasa.

Struktur Payload Permintaan (JSON)

{
  "apiKey": "fazzganzz",
  "amount": "1.500.000",
  "productName": "VIP Ticket Annual Pass",
  "buyerName": "Michael Scott",
  "buyerUsername": "mscott",
  "ownerName": "FazzPay Events",
  "ownerUsername": "fazzevents",
  "transactionId": "TXN-71937218319-FAZZ",
  "dateTime": "2026-07-01 19:45:00",
  "template": "glass"
}

Bentuk Respons

HTTP/1.1 200 OK
Content-Type: image/png
Content-Disposition: inline; filename="receipt.png"
