Integrasikan Pembuatan Struk Otomatis ke Sistem Bot Telegram / Web Anda

API FazzPayPic mengembalikan gambar PNG langsung secara dinamis melalui parameter HTTP GET query. Tidak perlu parsing file base64 yang rumit. Cukup kirim query, dapatkan gambar mentah instan.

Informasi Kunci API (Credentials)

Setiap endpoint wajib menyertakan parameter apikey. Kunci otentikasi utama hanya dipegang oleh owner untuk menghindari penyalahgunaan generator berlebihan. Hubungi kontak resmi di telegram @FazzGanteng untuk mengklaim API Key Anda sendiri.

Detail Endpoint Render Gambar

GET /api/generate

Parameter Query yang Diterima

Nama Parameter Tipe Status Deskripsi
apikey String Wajib (Required) API Key resmi yang valid, didapatkan melalui @FazzGanteng.
price Number Wajib (Required) Nominal harga transaksi (tanpa tanda titik/koma). Contoh: 75000.
buyer String Wajib (Required) Nama lengkap pembeli. Contoh: Supratman.
tx_id String Wajib (Required) Nomor referensi atau ID transaksi acak yang unik.
product String Wajib (Required) Nama barang atau jasa yang dibeli.
owner_name String Wajib (Required) Nama toko Anda / merchant utama.
owner_username String Opsional Username owner untuk kontak langsung di telegram/medsos.
date String Opsional Waktu transaksi spesifik. Default jika kosong adalah waktu render server.
template String Opsional Tema warna visual struk. Pilihan: blue (Default), emerald, violet.

Contoh Panggilan Integrasi

Menggunakan cURL (Command Line) 
curl -G "https://fazzpaypic.vercel.app/api/generate" \
  -d "product=VPS%20High%20Spec%201%20Bulan" \
  -d "price=150000" \
  -d "buyer=Fajar%20Santoso" \
  -d "tx_id=FP90181" \
  -d "owner_name=FazzStore" \
  -d "apikey=MASUKKAN_APIKEY_DISINI" \
  -o "bukti_transaksi_sukses.png"
Menggunakan Node.js / JavaScript Axios 
const axios = require('axios');
const fs = require('fs');

async function downloadReceipt() {
    const response = await axios({
        url: 'https://fazzpaypic.vercel.app/api/generate',
        method: 'GET',
        params: {
            product: 'Diamond Free Fire 1000 Pcs',
            price: 130000,
            buyer: 'Tono Wijaya',
            tx_id: 'TX-ID-908123',
            owner_name: 'FazzGanteng Shop',
            apikey: 'fazzganzz' // Ganti jika menggunakan API Key Anda
        },
        responseType: 'stream'
    });

    response.data.pipe(fs.createWriteStream('bukti_sukses.png'));
    console.log('Bukti transaksi berhasil diunduh ke disk!');
}

downloadReceipt();
