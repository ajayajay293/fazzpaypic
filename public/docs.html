FazzPay Receipt Generator API

API ini dirancang untuk menghasilkan gambar bukti transfer pembayaran digital yang aman, beresolusi sangat tinggi, dan tajam (1080x1920). Layanan ini sepenuhnya kompatibel dengan penyebaran Vercel Serverless.

⚠️ PERINGATAN KEAMANAN: Kunci API Anda bersifat pribadi. Harap jangan pernah menyebarkan parameter kunci API Anda pada kode repositori publik atau di sisi klien tanpa enkripsi yang mumpuni.

Spesifikasi Endpoint

POST /api/generate

Endpoint ini menerima data JSON transaksi dan mengembalikan aliran berkas gambar langsung berformat image/png.

Parameter Payload JSON

Parameter Tipe Deskripsi
apiKey string Wajib diisi dengan kunci yang sah: fazzganzz
amount string Jumlah nominal uang lengkap dengan satuan, contoh: "Rp 150.000,00"
productName string Nama barang atau layanan yang ditransaksikan.
buyerName string Nama lengkap pembeli / pengirim.
buyerUsername string Nama pengguna pembeli diawali tanda @ (opsional).
ownerName string Nama penerima dana (merchant).
ownerUsername string Nama pengguna penerima diawali tanda @ (opsional).
transactionId string Kode unik referensi transaksi perbankan.
dateTime string Waktu dan tanggal transaksi berlangsung.
template string Variasi gaya kartu. Pilihan: minimal, fintech, glass

Contoh Integrasi JavaScript (cURL)

fetch('/api/generate', {
  method: 'POST',
  headers: {
    'Content-Type': 'application/json'
  },
  body: JSON.stringify({
    apiKey: "fazzganzz",
    amount: "Rp 750.000,00",
    productName: "Sewa Server Cloud Pro",
    buyerName: "Doni Hermawan",
    buyerUsername: "@doni",
    ownerName: "Fazz Corp",
    ownerUsername: "@fazzcorp",
    transactionId: "TRX-99810-FP",
    dateTime: "01 Jul 2026, 16:02",
    template: "glass"
  })
})
.then(res => res.blob())
.then(blob => {
  const imageUrl = URL.createObjectURL(blob);
  document.getElementById('displayImage').src = imageUrl;
});

Respon Kode Kesalahan (Error Handling)

401 Unauthorized Diberikan jika Kunci API tidak sesuai atau tidak dikirimkan.
400 Bad Request Diberikan apabila salah satu kolom parameter utama diabaikan atau kosong.
